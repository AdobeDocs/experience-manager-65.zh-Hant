## 准則、秘訣與技巧 {#guidelines-tips-and-tricks}

* [SocialUtils重構](socialutils.md)  — 取代SocialUtils的SRP公用程式方法。
* [命名慣例](naming-conventions.md)  — 自訂Java類別的命名慣例。
