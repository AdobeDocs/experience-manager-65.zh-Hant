# EMC Documentum使用者的Connector備份策略 {#backup-strategy-for-connector-for-emc-documentum-users}

如果您已安裝Connector for EMC Documentum ，除了本章中的說明外，您的備份與復原策略還必須包括備份（或復原）安裝各個ECM系統的電腦。 （請參閱ECM Documentum檔案）。

使用ECM存放庫並執行下列工作，備份您的AEM表單環境：

* 依照本檔案所述指示備份AEM表單。
* 請依照下列說明備份您的ECM Documentum系統： [備份EMC Documentum Content Server](/help/forms/using/admin-help/backing-recovering-emc-documentum-repository.md#back-up-the-emc-documentum-content-server).

使用ECM存放庫並執行下列工作來還原AEM表單環境：

* 請依照以下說明還原個別的ECM系統： [還原EMC Documentum Content Server](/help/forms/using/admin-help/backing-recovering-emc-documentum-repository.md#restore-the-emc-documentum-content-server).
* 依照本檔案所述指示還原AEM表單。
