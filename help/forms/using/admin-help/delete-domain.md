# 刪除網域 {#delete-a-domain}

使用「域管理」頁可將現有域標籤為已過時。

如果建立的域與已刪除的域同名，則刪除的域將恢復，並包含該域所包含的資訊。 無法刪除DefaultDom網域。

1. 在管理控制台中，按一下 **[!UICONTROL 設定>使用者管理>網域管理]**.
1. 選中要刪除的域的複選框，按一下 **[!UICONTROL 刪除]**，然後按一下 **[!UICONTROL 確定]**.
