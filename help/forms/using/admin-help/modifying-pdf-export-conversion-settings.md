# 修改PDF導出轉換設定 {#modifying-the-pdf-export-conversion-settings}

使用以下過程修改導出PDF、EPS、DOC、TXT、RTF、XML或HTML檔案的轉換設定。 預設情況下，PDF檔案使用在Adobe Acrobat專業版或Acrobat Standard版中配置的預設「另存為」設定。 例如，在Acrobat中將PDF檔案轉換為EPS的預設「另存為」設定將只導致將PDF檔案轉換為EPS的頁面。

>[!NOTE]
>
>在修改一種檔案格式的「另存為」設定後，它將應用於從「PDF生成器」導出的相同類型的所有轉換。

1. 在Acrobat中開啟PDF檔案時，選擇「檔案」>「另存為」。
1. 在「另存為類型」(Save As Type)清單中，選擇相應的格式。
1. 按一下「設定」(Settings)並根據需要設定檔案格式設定。
1. 按一下「確定」，然後按一下「保存」以導出PDF檔案。
