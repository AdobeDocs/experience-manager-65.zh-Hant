# PDF Generator備份限制 {#pdf-generator-backup-limitations}

無法備份PDF Generator用來轉換檔案的暫存目錄。 即使服務已正確還原，資料仍可能會遺失，因為PDF Generator會以設定的間隔檢視及清除暫存目錄的內容。
