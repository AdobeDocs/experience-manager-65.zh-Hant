# PDF生成器備份限制 {#pdf-generator-backup-limitations}

無法備份PDF生成器用於轉換檔案的臨時目錄。 即使服務將正確恢復，資料也可能會丟失，因為PDF生成器會按設定的時間間隔查看和清除臨時目錄的內容。
