# 搜尋使用者或群組 {#search-for-a-user-or-group}

>[!NOTE]
> 
> 確保使用者具有存取管理員控制檯的管理員許可權。

1. 在管理控制檯中，按一下設定>使用者管理>使用者和群組。
1. 指定資訊以縮小搜尋範圍，然後按一下「尋找」。 搜尋結果會列在頁面底部。 您可以按一下任何欄標題來排序清單。 最多會傳回1000個結果。
