# 搜索用戶或組 {#search-for-a-user-or-group}

1. 在管理控制台中，按一下「設定」>「用戶管理」>「用戶和組」。
1. 指定資訊以縮小搜索範圍，然後按一下「查找」。 搜索結果列在頁面底部。 通過按一下任何列標題，可以對清單進行排序。 最多返回1000個結果。
