# 搜索用戶或組{#search-for-a-user-or-group}

1. 在管理控制台中，按一下「設定>使用者管理>使用者和群組」。
1. 指定要縮小搜索範圍的資訊，然後按一下「查找」。 搜尋結果會列在頁面底部。 您可以按一下任何欄標題來排序清單。 最多會傳回1000個結果。
