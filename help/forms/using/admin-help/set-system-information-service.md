# 設定系統資訊服務 {#set-up-the-system-information-service}

系統資訊服務提供REST API以檢索資訊。 要使用系統資訊服務，請從管理控制台啟用REST終結點。 執行以下步驟以啟用REST端點：

1. 登錄到管理控制台。 管理控制台的預設URL為 `https://[hostname]:'port'/adminui.`
1. 定位至「服務」>「應用程式和服務」>「服務管理」。
1. 在「服務管理」頁上，按一下 **系統資訊** 服務。
1. 在「端點」(Endpoints)頁籤的清單中，選擇「REST」(REST)，然後按一下 **添加**。
1. 在「添加REST端點」螢幕上，按一下 **添加**。
