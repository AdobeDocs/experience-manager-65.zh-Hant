# 從右到左的HTML語 {#right-to-left-languages-in-html-forms}

HTML5表單支援從右到左的語言，如希伯來語。 您可以以從右到左的語言顯示和填充HTML5表單。 您是從右到左和從左到右語言，形式相同。 例如，可以在公共欄位中同時使用英語和希伯來語。 移動Forms使用區域設定在表單中顯示本地化的日期和時間資訊。

**限制**

移動Forms在除Internet Explorer之外的所有瀏覽器中都支援從右到左的語言。
