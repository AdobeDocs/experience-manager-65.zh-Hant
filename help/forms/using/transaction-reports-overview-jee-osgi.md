# 適用於AEM Forms的Transaction Reporting {#transaction-reports-overview}

AEM Forms中的交易報告可讓您監視自AEM Forms部署上的指定日期以來的所有交易。 此功能旨在取得表單使用的相關資訊，以協助業務利害關係人深入瞭解其數位處理量。 在此上下文中，交易包含各種活動，例如：

* 提交最適化表單、HTML5表單或表單集
* 互動式通訊的列印或網頁版本轉譯
* 表單的提交和轉譯
* 將檔案從一種檔案格式轉換為另一種檔案格式
