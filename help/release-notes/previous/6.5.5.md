# [!DNL Adobe Experience Manager] 6.5.5.0 {#experience-manager-6550}

Adobe Experience Manager6.5.5.0是一個重要的更新，包括新功能、客戶要求的關鍵增強功能以及效能、穩定性和安全性改進，這些功能自6.5版在2014年12月正式上市以來發佈 **2019年4月**。 可安裝在Adobe Experience Manager6.5的上面。

中介紹的一些關鍵功能和增強功能 [!DNL Adobe Experience Manager] 6.5.5.0包括：

* 不允許匿名訪問CRXDE Lite。 而是將用戶定向到登錄螢幕。 請參閱 [用CRXDE Lite開發](/help/sites-developing/developing-with-crxde-lite.md)。

* 可以自定義 [!DNL Experience Manager] 「收件箱」：更改列的預設標題，對列的位置重新排序，並根據工作流的資料顯示其他列。 成員 `administrators` 或 `workflow-administrators` 組可以自定義列。 有關詳細資訊，請參見 [管理控制](/help/sites-authoring/inbox.md#inbox-admin-control)。

![自定義Experience Manager收件箱列](/help/release-notes/assets/customize-columns.gif)

* 在Experience ManagerWeb內容管理(WCM)的各個領域（如頁面編輯器、核心元件、RTE和管理用戶介面）中改進了可訪問性。

* 您可以使用代理UI為每個互動式通信保存一個或多個草稿，稍後檢索草稿以繼續處理。 可以為每個繪製指定不同的名稱以標識它。 有關詳細資訊，請參見 [將互動式通信另存為草稿](/help/forms/using/prepare-send-interactive-communication.md#save-as-draft)。

![另存為草稿](/help/release-notes/assets/save-as-draft.gif)

* 支援 [!DNL Oracle WebLogic 12] Experience Manager Forms的JEE。

* 改進的異常處理 [!DNL Adobe Experience Manager Assets] 用戶介面流。

* 獲取Dynamic Media·Scene7的發佈URL，一種新方法 `getRemoteAssetPublishURL` 添加到 `com.day.cq.dam.api.s7dam.scene7.ImageUrlApi` 。

* [輔助功能增強](#assets-6550) 在 [!DNL Adobe Experience Manager Assets] 符合Web內容輔助功能指南(WCAG)。

* 已從Adobe Experience Manager內刪除包共用整合。

* 內建存放庫 (Apache Jackrabbit Oak) 更新至 1.22.3 版。

以下是中提供的修復清單 [!DNL Experience Manager] 6.5.5.0版。

## UberJar {#uber-jar}

UberJarExperience Manager6.5.5.0在 [Maven中央儲存庫](https://repo.maven.apache.org/maven2/com/adobe/aem/uber-jar/6.5.5/)。

要在Maven項目中使用UberJar，請參見 [如何使用UberJar](/help/sites-developing/ht-projects-maven.md) 並在項目POM中包括以下依賴關係：

```shell
<dependency>
     <groupId>com.adobe.aem</groupId>
     <artifactId>uber-jar</artifactId>
     <version>6.5.5</version>
     <scope>provided</scope>
</dependency>
```

>[!NOTE]
>
>UberJar和其他相關工件可在Maven Central Repository（Maven中央儲存庫）上找到，而不是AdobePublic Maven儲存庫(`repo.maven.apache.org`)。 主UberJar檔案已更名為 `uber-jar-<version>.jar`。 所以，沒有 `classifier`, `apis` 值，對於 `dependency` 標籤。
