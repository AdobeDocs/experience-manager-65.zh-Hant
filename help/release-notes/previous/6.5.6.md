# [!DNL Adobe Experience Manager] 6.5.6.0 {#experience-manager-6560}

Adobe Experience Manager6.5.6.0是一個重要的更新，包括新功能、客戶要求的關鍵增強功能以及效能、穩定性和安全性改進，這些功能自6.5版在2014年12月正式上市以來發佈 **2019年4月**。 可安裝在Adobe Experience Manager6.5的上面。

Adobe Experience Manager推出的主要特點和增6.5.6.0包括：

* 有選擇地將資產發佈或取消發佈到 [!DNL Experience Manager] 或 [!DNL Dynamic Media] 使用 [!UICONTROL 快速發佈] 或 [!UICONTROL 管理發布] 的子菜單。

* 使用 [!DNL Dynamic Media] 使內容分發網路(CDN)快取內容失效的用戶介面。

* 現在，通過代理伺服器也支援將資產貢獻資料夾從Brand Portal發佈到Experience Manager Assets。

* 與資料夾（專用和非專用）關聯的用戶組現在將從上的儲存庫中刪除 [刪除這些資料夾](/help/assets/private-folder.md#delete-private-folder)。 但是，可以使用JMX從儲存庫中刪除現有的冗餘、孤立、未使用和自動生成的用戶組。

* 視頻中修飾符的說明 [!UICONTROL 查看器] 已在中更新預設編輯器 [!DNL Dynamic Media]。

* 提供新公司設定以反映 [!DNL Dynamic Media] 連接器。

* 預設選項 `test` 和 `aiprocess` 已更新為 `Thumbnail`從 `Rasterize` 以前在Dynamic Media，以確保用戶只需建立縮略圖並跳過頁面提取和關鍵字提取。

* [在客戶端預填自適應表單](/help/forms/using/prepopulate-adaptive-form-fields.md#prefill-at-client)。

* [在具有雙向SSL實現的伺服器上與REST風格API整合表單資料模型](/help/forms/using/configure-data-sources.md)。

* [用於翻譯的自適應表單頁的增強的快取](/help/forms/using/configure-adaptive-forms-cache.md)。

* 支援 [Adobe SignAutomated forms conversion服務中的文本標籤](https://experienceleague.adobe.com/docs/aem-forms-automated-conversion-service/using/convert-existing-forms-to-adaptive-forms.html)。

* 支援 [將彩色表單轉換為自適應表單](https://experienceleague.adobe.com/docs/aem-forms-automated-conversion-service/using/convert-existing-forms-to-adaptive-forms.html) 使用 [!DNL Automated Forms Conversion service]。

* 支援SMB 2和SMB 3協定。

* 內建存放庫 (Apache Jackrabbit Oak) 更新至 1.22.4 版。

以下是中提供的修復清單 [!DNL Experience Manager] 6.5.6.0版。
