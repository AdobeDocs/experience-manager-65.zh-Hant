# 工具主控台{#tools-consoles}

**「工具」**&#x200B;主控台可供存取多種特殊工具，協助您管理網站、數位資產和內容存放庫的其他方面。**Tools**&#x200B;控制台目前有兩種類型，取決於您使用的UI:

* [工具——觸控最佳化UI](#tools-touch-optimized-ui)
* [工具- Classic UI](#tools-classic-ui)
