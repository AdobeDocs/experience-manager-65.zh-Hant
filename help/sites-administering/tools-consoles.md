# 工具主控台{#tools-consoles}

**「工具」**&#x200B;主控台可供存取多種特殊工具，協助您管理網站、數位資產和內容存放庫的其他方面。目前有兩種 **工具** 控制台取決於您使用的UI:

* [工具 — 觸控優化UI](#tools-touch-optimized-ui)
* [工具 — 經典UI](#tools-classic-ui)
