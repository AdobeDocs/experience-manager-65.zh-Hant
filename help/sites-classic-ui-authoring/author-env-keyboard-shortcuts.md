# 控制台的鍵盤快捷鍵{#keyboard-shortcuts-for-consoles}

您可以使用各種鍵盤快捷鍵AEM。 有些適用於控制台的使用，有些適用於 [頁面編輯](/help/sites-classic-ui-authoring/classic-page-author-keyboard-shortcuts.md)。

>[!NOTE]
>
>案頭設備上的多選：
>
>* 窗口： **Ctrl鍵**+**按一下** 所需元件。
>
>* Mac: **命令**+**按一下** 所需元件。
>
