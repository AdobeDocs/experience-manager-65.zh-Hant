# 提供動態媒體資產{#delivering-dynamic-media-assets}

您如何提供動態媒體資產（包括視訊和影像）取決於網站的實作方式。

使用動態媒體時，您有數個選項：

* 如果您的網站是由AEM代管，則您想要直接將動態媒體資產新增至您的頁面。
* 如果您的網站未在AEM上，請參閱 [在觸控式使用者介面中的「傳送動態媒體資產](/help/assets/delivering-dynamic-media-assets.md) 」。

如需詳細資訊，請參閱下列主題：

* [新增動態媒體資產至頁面](/help/sites-classic-ui-authoring/dynamic-media-assets-adding-to-page.md)
