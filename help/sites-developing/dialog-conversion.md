# 對話方塊轉換工具{#dialog-conversion-tool}

提供的對話方塊轉換工具可協助擴充僅針對傳統UI （根據ExtJS）或根據Granite UI和Coral 2定義對話方塊的現有元件。 此工具會使用原始對話方塊，根據Granite UI和Coral 3，建立為標準UI設計的重複對話方塊。

對話方塊轉換工具是 [Adobe現代化工具套裝](modernization-tools.md).
