# 對話框轉換工具{#dialog-conversion-tool}

提供對話框轉換工具以幫助擴展僅為經典UI（基於ExtJS）或基於花崗岩UI和Coral 2定義對話框的現有元件。 該工具使用原始對話框建立一個基於花崗岩UI和Coral 3的標準UI設計的重複對話框。

對話框轉換工具是 [Adobe現代化工具套件。](modernization-tools.md) 有關詳細資訊，請參閱該文檔。
